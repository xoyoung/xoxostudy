html문서는 크게 2가지 형태를 가짐

  1. block : 크기를 가지고, 내용이 줄바꿈처리
  2. inline : 크기를 가질 수 없고, 내용이 옆으로 나열

인라인 요소의 대표 코드는
강조를 표현하는 기능들이 많다.

강조 이의에도 많은 인라인 요소들이존재
많이 사용하는 기능중 링크의 기능,

이미지 삽입기능이 있다.

inline 형태의 특성은 반드시 block 형태의 요소 내부에 존재
